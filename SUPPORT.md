# Getting Help

## Questions about the data or methodology?
Open a [GitHub Discussion](https://github.com/thedatavigilante/UI_INDEX/discussions) or open an issue using the [Feature Request template](.github/ISSUE_TEMPLATE/feature_request.yml).

## Found a data error?
Please open an issue using the [Data Correction template](.github/ISSUE_TEMPLATE/data_correction.yml). Include the jurisdiction, the current value, the expected value, and a link to the authoritative source.

## Want to reuse this analysis for another state?
See [CONTRIBUTING.md](CONTRIBUTING.md) for guidance on extending the pipeline. The indices are designed to be jurisdiction-agnostic — you only need to swap in your state's benefit caps, wage bases, and housing costs.

## Press or policymaker inquiries?
Email with subject line: `[UI_INDEX Press Inquiry]` — see contact details in [about.html](https://thedatavigilante.github.io/UI_INDEX/about.html).

## Security vulnerability?
See [SECURITY.md](SECURITY.md) for reporting instructions. Do not open a public issue.
